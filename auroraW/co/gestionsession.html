Gestion de session d'utilisateur

Certains navigateur comme Chrome permettent de gérer un profil d'utilisateur à partir d'un compte email. Cela permet notamment de récupérer sur un autre poste les paramètres de son navigateur, les favoris, ses extensions et plugin et d'autres réglages.

Cette fonctionnalité est très intéressante quand vous avez plusieurs appareils avec le même compte, vous pouvez donc synchroniser le profil utilisateur sur ces différents postes.
